#Future is now!
Start creating projects using EcmaScript 6 and AngularJS and whatever-you-want-more!

#Requirements
* npm
* webpack (installed globally)

#Quick start
To build dependencies just run:
```she
npm install
```
and then build bundle.js:
```
webpack
```
or to build and listen for changes:
```
webpack --watch
```
